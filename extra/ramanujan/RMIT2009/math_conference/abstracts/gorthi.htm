Model Based Generation of Software Test Cases and Regression Test Cases

In this talk, the speaker presents, 

i)        An overview on Test Case and Regression Test Case Creation and Maintenance

a.       Test Plan vs Test Execution

b.      Creation and maintenance of test scenarios, cases, data and scripts

c.       What are regression test cases? Creation and maintenance of regression test cases

ii)       The relevance of this area to the Total-Cost-of-Ownership (TCO) of medium to large s/w applications

a.       Typical number of test scenarios and cases in a medium / large s/w application

b.      Efforts and costs involved in the creation and maintenance of test cases and regression test cases

iii)     The state-of-the-art research and practices of this field

a.       Model based (UML Activity Diagram, State Diagram based) manual and automated creation and maintenance of test and regression test cases

iv)     Some interesting work done by the speaker and his team in this area for the last three years

a.       The concept of ‘Unit of Behavior’

b.      Techniques for automatic generation of test cases and regression test cases from UML Activity Diagrams

c.       Results of field trials of these techniques
