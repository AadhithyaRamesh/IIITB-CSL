Devnagari Enhanced Keyboard Layout

Features

  • Color Scheme To Narrow down the search for letters on Keyboard
  • Easy to Learn
  • Supports UNICODE.
  • Layout is according to specifications standardised by Govt. Of India.

The Keyboard allows user to locate Devanagri letters on Keyboard fast, which is not possible with normal single color keyboard.

Each VARGA (for e.g. क वर्ग contains क ख ग घ ङ ) is assigned a unique color and correspondingly, keys on keyboard are mapped using same color scheme.

This color scheme narrows down the search for letter on keyboard and helps user to identify any VARNA (letters) quickly.
